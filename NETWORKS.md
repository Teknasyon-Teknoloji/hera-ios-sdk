# Ad Networks Adapters

  

Hera comes with Admob and Facebook ads at its core, but you can optionally integrate 5 other networks as well. Here is a list of the the networks

# IronSource

### Integration  

 `pod HeraSDK/IrounSource`

### Network Identifiers

Please add the following keys into Info.plist if you want to use this network

``` xml
<key>SKAdNetworkItems</key>
<array>
<dict>
<key>SKAdNetworkIdentifier</key>
<string>su67r6k2v3.skadnetwork</string>
</dict>
</array>
```
